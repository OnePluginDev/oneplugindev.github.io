AggieSync

Unofficial course catalog viewer for A&M students. Mirrors most of the functionality with added GPA distributions, updated design, and full section/course loading.

While anyone can use this plugin and view courses, you must use the registration button on Howdy to actually sign up for a class.

Requires Google Chrome with Web Store access to install.
